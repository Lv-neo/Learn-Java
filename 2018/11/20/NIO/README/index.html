NIO 入门

JDK 1.4 中引入的新输入输出 (NIO) 库在标准 Java 代码中提供了高速的、面向块的 I/O。

本实用教程从高级概念到底层的编程细节，非常详细地介绍了 NIO 库。

您将学到诸如缓冲区和通道这样的关键 I/O 元素的知识，并考察更新后的库中的标准 I/O 是如何工作的。

您还将了解只能通过 NIO 来完成的工作，如异步 I/O 和直接缓冲区。
